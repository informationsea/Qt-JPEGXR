Qt-JPEGXR
=========

Qt Image Plugin for JPEG XR

License
-------

LGPL 3 or later

Limitations
-----------

This plugin is under initial development.

* Quality of image is fixed at 82
  * If you want to change, please refer jpegxrhandler.cpp, line 189
* No alpha support
* Supports only BGR32, no other color space
